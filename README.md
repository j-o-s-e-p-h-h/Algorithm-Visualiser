## Run

```bash
python visualizer.py
```

## Controls

- `R`: reset the list
- `SPACE`: start or pause the current sort
- `A`: sort in ascending order
- `D`: sort in descending order
- `B`: Bubble Sort
- `I`: Insertion Sort
- `S`: Selection Sort
- `C`: Cocktail Sort
- `M`: Merge Sort
- `Q`: Quick Sort
