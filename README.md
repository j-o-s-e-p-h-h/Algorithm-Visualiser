# Algorithm Visualiser

A polished Pygame sorting visualizer with animated bars, live controls, and multiple classic algorithms.

Run it with:

```bash
python visualizer.py
```

Controls:

- `SPACE`: start or pause the current sort
- `R`: reshuffle the list
- `A` / `D`: switch ascending or descending order
- `B`: Bubble Sort
- `I`: Insertion Sort
- `S`: Selection Sort
- `C`: Cocktail Sort
- `M`: Merge Sort
- `Q`: Quick Sort
- `UP` / `DOWN`: change the number of bars
- `LEFT` / `RIGHT`: change the animation speed
